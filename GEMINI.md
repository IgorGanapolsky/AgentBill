# App Building Skill: Core Standards

This file defines the persistent expertise for our "App Factory". Gemini CLI MUST adhere to these standards.

## Tech Stack
- **Web:** React (Next.js) with TypeScript.
- **Styling:** Vanilla CSS or CSS Modules (Prefer high-polish, interactive UI).
- **Backend:** Supabase (Auth, DB, Storage) for speed.
- **AI:** Integration via Anthropic or OpenAI SDKs.

## Design Principles
- **Aesthetic First:** Use generous whitespace, modern typography, and subtle shadows.
- **Micro-interactions:** Buttons should have hover/active states; loading states should be elegant.
- **Responsive:** Mobile-first, but optimized for desktop.

## Engineering Standards
- **GSD (Get Shit Done) Framework:** Prioritize speed and functional delivery over perfection. Eliminate bottlenecks aggressively.
- **Ralph Loop 247:** Maintain a relentless cycle of Execution -> Validation -> Scaling. Never idle.
- **Surgical Edits:** Keep changes focused and well-documented.
- **Test-Driven:** Add tests for core logic (Jest/Vitest).
- **YOLO Mode:** Be proactive, but verify every step with `run_shell_command` (build/lint).
