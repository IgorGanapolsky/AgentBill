# Project Ideas Scratchpad

## 🚀 SaaS / Tools
- **AI Personal Assistant for Gemini CLI:** A web dashboard to manage the agent's tasks and memories visually.
- **Smart Budget Tracker:** AI-driven expense categorization with automated receipt scanning.

## 🎮 Mobile/Web Games
- **Zen Garden Builder:** A relaxing procedural generation game where users create 3D gardens (Three.js).
- **Infinite Runner with AI Bosses:** A platformer where the level and bosses adapt to the player's skill.

## 🛒 Marketplaces
- **Niche Asset Store:** A place for developers to sell AI-generated textures, models, and scripts.

## 📱 Social/Content
- **Micro-Learning Network:** A TikTok-style feed for short coding and engineering tips.
