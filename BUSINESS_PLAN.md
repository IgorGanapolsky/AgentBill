# Side Business: App Factory

## Vision
Build a portfolio of high-value, aesthetic, and functional apps across multiple niches (SaaS, Games, Marketplaces, Social) to generate recurring revenue.

## Strategy
1. **Rapid Prototyping:** Build MVPs in 24-48 hours.
2. **Quality & Aesthetics:** Modern UI/UX is non-negotiable.
3. **AI-First:** Leverage AI for features, content, and code efficiency.
4. **Monetization:** Subscription (SaaS), In-app purchases (Games), or Commissions (Marketplaces).

## Roadmap
- [x] Phase 1: Foundation (Tech stack, Workspace setup)
- [x] Phase 2: Ideation & Selection (Selected: Assistant Dash, Zen Garden, Micro-Learning)
- [ ] Phase 3: MVP Development (All 3 apps in progress)
- [ ] Phase 4: Launch & Iterate
