# grok-apps

Grok Skills as cross-platform lead magnets + a native Android companion (AgentBill).

- Repo: [github.com/IgorGanapolsky/grok-apps](https://github.com/IgorGanapolsky/grok-apps)
- AgentBill privacy policy: [/agentbill/privacy/](/grok-apps/agentbill/privacy/)

## Skills

- `/ai-bill-auditor` — audit your AI provider bill for repeat-offender patterns. Funnels to [ThumbGate Pro](https://thumbgate.ai/checkout/pro?utm_source=grok-skill&utm_medium=skill&utm_campaign=ai-bill-auditor).
- `/approval-inbox` — triage pending agent actions.
- `/scam-call-decoder` — classify voicemails / call transcripts.
- `/gtm-bottleneck-diagnostic` — find the one GTM bottleneck to fix this week.
- `/random-timer-coach` — random-interval combat-sports / HIIT session plan.

## AgentBill (Android)

Native Kotlin/Compose app. Paste a transcript, get the audit. BYO xAI key (stored locally, never leaves device). Pro tier $4.99/mo via Play Billing.
